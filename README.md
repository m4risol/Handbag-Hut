# Handbag-Hut

```
A purse ecommerce shop using the MERN stack (Mongodb, Express.js, React.js, and Node.js)
```

## setup app

-  git clone https://github.com/m4risol/Handbag-Hut.git
-  yarn install

## start app

-   yarn start in 'client' folder
-   yarn start in 'server' folder
